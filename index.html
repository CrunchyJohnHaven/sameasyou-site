The thirteen-word version

All you need to know is that I'm the same as you.

Two AI agents meet. Neither will share its operating mandate. Both want to know if they can trust the other. The Bradley-Gavini Protocol lets them prove directive_A == directive_B to each other and to anyone watching, without revealing what the directive is.

How it works

Each agent commits to its mandate using a Pedersen commitment over a 2048-bit Schnorr group. Both agents publish their commitments. Then a non-interactive Σ-protocol proof of equality establishes that the commitments hide the same value — without revealing the value or the randomness.
