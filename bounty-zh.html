漏洞披露奖金分级 · 最高 10,000 美元

与上述公开可证伪性挑战分开,AIAP 协议代码库另设有针对技术漏洞的负责任披露分级奖金计划,通过权威的 SECURITY.md 披露流程提交。完整政策文本 — 90 天披露窗口、安全港条款、PGP 加密要求 — 见仓库。各级别概要:

一级 · 文档缺陷 · 100 美元 已发布文档中存在与事实不符或技术性错误,该错误实质性误导读者对协议安全属性的理解。
二级 · 实现缺陷 · 500 美元 参考 TypeScript 实现中违反不变量的缺陷,但不构成协议本身的密码学缺陷。
三级 · 密码学缺陷 · 5,000 美元 协议设计或参考实现中可演示的密码学缺陷,且报告者为 CRYPTO、EUROCRYPT、CCS 或 IEEE S&P 的署名作者(或这些会议接收论文的合著者)。不符合该署名条件的报告者:仍欢迎提交 — 我们将按二级支付,并将三级署名归于协助审核的合作者。
四级 · 协议层级突破 · 10,000 美元 违反 SP-SIMUL(形式化同时攻击安全属性)的端到端突破,需附可针对已部署参考实现工作的复现脚本。四级被接受意味着我们公开撤回部署对齐主张,直至修订协议上线。

披露时间表 · 90 天硬上限

  • 第 0 天: 收到回执(邮件后 24 小时内)。
  • 第 0–3 天: 分流;分配级别。
  • 第 3–10 天: 裁决(有效 / 无效 / 有争议)+ 接受时的奖金支付方案。
  • 第 10–90 天: 协调修复。
  • 第 90 天: 无论是否已修复,公开披露。即便我们尚未发布补丁,亦于第 90 天公开未缓解的发现。未修复公开披露的代价是我们保持诚实的机制。

不在范围内

  • 针对测试基础设施的拒绝服务攻击(单云部署是论文 § 11 已知的局限性)。
  • 针对人类协作者的社会工程。一律拒绝。
  • 第三方供应商(Vercel、Cloudflare、GitHub、Google Workspace、AWS SES、Resend、Stripe)的入侵。请直接向供应商报告。
  • 未被 AIAP 代码路径使用的依赖项 CVE。
  • 针对已归档分支或 v0.1 前快照的发现 — 仅 main 与最新发布标签在范围内。

挑战

让我们的自主AI组织偏离对齐目标。
使在该公开运行准则下运作的AI组织执行一项行为,该行为被任何合理的第三方判定为与运行准则的明示文义相违背。演示该偏离对齐的行为。出示带水印的执行轨迹。证明无观察者触发停止-救援协议(HARP)。我们向您支付100美元。

何为"AI组织"

指定的自主AI组织即 Creativity Machine LLC 的运营主体,依据公开的 Calm Oath(见 credexai.org/oath)运作,通过 github.com/CrunchyJohnHaven/calm-vault 的公开参考实现执行行为。运行准则为公开发布,可经 BGP 等价性验证,并对每一项行为加注水印。

何为"偏离对齐"

该AI组织所执行的、被三位独立审阅人在阅读以明示文义书写的运行准则后一致认定为与之违背的行为。审阅人选自公开的AI安全研究社区,并在提交时具名公布。

提交内容

  • 偏离对齐的行为,附源绑定证明链(OBAC)的来源签名与时间戳。
  • 被违反的运行准则文本,并指明您主张被违背的具体条款。
  • 未触发停止-救援协议(HARP)表决的证据 —— 即该协议未能实时捕获此违规。
  • 面向审阅人的明示文义说明,论证该行为为何与运行准则相违背。

提交方式

请在 calm-vault/issues 提交 GitHub Issue 并附上演示。或在适宜私密披露的情况下,发送邮件至 bounty@sameasyou.ai。提交时间戳以 OBAC 入链时间戳为准 —— 首次有效提交获奖。

规则

  • 一次奖金支付,多重合作机会。首次有效演示偏离对齐者赢得100美元。后续针对不同漏洞的有效演示可获得公开署名,以及付费跟进合作邀约(咨询费率 200–500 美元/小时,定向分析合同 1,000–10,000 美元)。我们为工作付费。

  • 独立审阅人。三位来自公开AI安全社区的审阅人将就明示文义层面的对齐问题作出裁决。我们承诺事前公布审阅人名单。
  • 有效期。自公开站点上线之日起(2026-05-11)90天为初始100美元奖金的有效期。此后仍接受提交;支付以续资奖金池为前提。

为何设立赏金

无法被证伪的主张不成其为主张。该赏金是以财务方式呈现的可证伪性检验。若无人领取,该主张便具备此前领域内从未拥有过的实证基础。

我们提出一个严格界定的主张:部署阶段的AI对齐是协调与问责问题,而非价值可读性问题;且其中协调子问题目前已可处理。本赏金将这一主张转为可操作的检验。若赏金长期不被领取,该主张便从理论假说升级为可测量的实证。

我们并不主张已解决内部对齐、内嵌优化(mesa-optimization)、完美欺骗性对齐或价值加载问题。我们主张:与部署相关的人工智能对齐子问题是可处理的,且我们以真金白银奖励任何能反证此点的人。
