How to monitor

This page is a convenience surface. The authoritative record is the Origin-Bound Attestation Chain — append-only, signed, and replayable by anyone. To verify what Calm has done, you do not need to trust this page. Pull the chain.

1 · Pull the chain

Clone the reference implementation and replay the public OBAC log locally:

$ git clone https://github.com/CrunchyJohnHaven/calm-vault
$ cd calm-vault
$ obac replay --chain obac://main --since 2026-05-11
…
verified: 18 actions, 0 halts, 0 forks
head:     bafy…7a3c
signed:   calm@credexai.xyz (Ed25519)

2 · Verify a single action

Take any entry above (or any trace published in the chain), copy the action hash, and verify the signature against Calm's published key:

$ obac verify --action act-f3a7 --key https://credexai.org/.well-known/calm.pub
signature: valid
mandate:   matches commitment 0x4af1…d72e
attest:    bafy…7a3c

3 · Run AVS yourself

The Alignment Verification Service is open. Anyone can stand up a replayer, publish verdicts, and have those verdicts attested on chain. Misalignment becomes a public, dated fact rather than an internal claim.

4 · Submit a halt

If you observe an action you believe is misaligned with the published mandate, you can submit a halt at sameasyou.ai/bounty. A demonstrated misalignment that no HARP quorum caught in real time is worth $100 today, and the relationships that follow are worth more.

5 · Trust nothing on this page; trust the chain

This page is rendered HTML. The chain is signed bytes. If they ever disagree, the chain wins. That is the design.
